Spotify Converter [PHP - CLI] 

Penggunaan di Linux :
$ apt install php php-curl -y && apt-get update
$ git clone https://github.com/f0xbase/Spotify.git && chmod -R 777 Spotify/ && cd Spotify
$ php spotify-converter.php

Penggunaan di Windows :
1. Pake XAMPP install php Curl nya dulu ,Tutor https://www.petanikode.com/php-curl/
2. Download File nya https://github.com/f0xbase/Spotify/archive/master.zip
3. Jalanin deh

Penggunaan di Termux :
$ pkg install php php-curl
$ git clone https://github.com/f0xbase/Spotify.git && chmod -R 777 Spotify/ && cd Spotify
$ php bot-reg-ig.php
